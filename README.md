# React Material TS Template

### Install Yarn and Node
1. install [yarn](https://classic.yarnpkg.com/en/docs/install/)

2. install [node](https://nodejs.org/en/download/)

### Install dependencies
```
yarn install
```

### Start app
```
yarn start
```

### Test app
```
yarn test
```

### Lint app
```
yarn lint
```
